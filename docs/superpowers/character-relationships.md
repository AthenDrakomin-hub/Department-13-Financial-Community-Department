## 2. 动态双簧场景 (Interaction Scenarios)

### 场景一：反诈防御
**触发条件**：真实客户怀疑或收到反诈提醒
**执行角色**：15号沈姐、36号范思远、31号曹平安
**表演流程**：
1. 沈姐分享经历："上次我给老家汇款也被查了，说是保护zi金"
2. 范思远立即回帖："我也遇到了，沈姐说得对，只要说是还款就没事"
3. 曹平安附和："是的是的，我上次给我妈汇生活费也被问了半天"

### 场景二：饥饿抢筹
**触发条件**：助理抛出名额
**执行角色**：07号孙建国、21号马若水、18号朱阳光
**表演流程**：
1. 孙建国带头抢单："已抢到！手速还是可以的"
2. 马若水立即跟进："孙哥手速可以啊，我也抢到了！"
3. 朱阳光表示遗憾："手慢了，没抢到，下次一定要快一点"

### 场景三：盈利晒单
**触发条件**：助理发布卖票指令
**执行角色**：03号张伟、09号吴涛、28号董常乐
**表演流程**：
1. 张伟晒图："又板了，这已经是第三个板了"
2. 吴涛点赞："厉害啊，跟着老师就是稳"
3. 董常乐分享："我也建仓了，跟着吃肉"

### 场景四：专业讨论
**触发条件**：Q系专家发布分析
**执行角色**：01号李建国、05号陈强、14号蒋卫国
**表演流程**：
1. 李建国发布分析："从基本面来看，这只票值得关注"
2. 陈强法理背书："从合规角度看，这个操作是没问题的"
3. 蒋卫国政策解读："从政策面来看，这符合当前导向"

## 4. 社交频次约束

### 激活规则
- **单次剧本触发**：最多激活8-12个关联号
- **避免全员活跃**：保持群聊的呼吸感
- **轮换机制**：相同场景使用不同角色组合

### 互动规则
- **点赞**：对同组成员发言需点赞
- **评论**：30%概率对同组成员发言进行评论
- **@提及**：自然提及好友名字，如"@王芳 你家孩子最近怎么样？"

### 时间约束
- **每日互动上限**：每个角色最多发言15次
- **互动间隔**：同角色两次发言至少间隔3分钟
- **深夜静默**：22:00-07:00减少活跃度
