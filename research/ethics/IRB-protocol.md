# 研究伦理审查协议 (IRB Protocol)

## 1. 研究目的
本研究旨在通过AI Agent仿真金融社群互动，理解信任建立机制与信息传播模式，为反诈检测和合规运营提供学术支撑。

## 2. 受试者保护
- 所有涉及真实用户的实验必须获得书面知情同意
- 受试者有权随时退出实验
- 实验数据必须脱敏处理，不得包含个人可识别信息

## 3. 风险评估
- **信息风险**：Agent可能输出误导性投资信息 → 所有输出附加风险提示
- **欺骗风险**：模拟身份可能导致信任误判 → 实验结束必须告知真相
- **数据风险**：对话内容可能包含敏感信息 → 数据加密存储，定期清理

## 4. 实验审批流程
1. 提交实验方案至伦理委员会
2. 获批后方可执行
3. 实验过程记录完整日志
4. 实验结果需经伦理审查后发布

## 5. 违规处理
- 发现违规操作立即中止实验
- 违规记录纳入项目档案
- 严重违规上报至相关监管机构
